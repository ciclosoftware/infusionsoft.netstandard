To setup authorization...

1. Go to this Infusionsoft page

2. Press 'API Access' next to the account you want to use

3. Select the 'Partner' tab

4. Enter the client_id

5. Press 'Generate Auth Code'

6. Copy the auth code here:
7. Press 'Authorize' to setup auth in the backend

This authorization could also happen with a popup dialog showing you Infusionsoft.
